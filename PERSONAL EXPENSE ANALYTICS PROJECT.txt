PERSONAL EXPENSE ANALYTICS PROJECT DOCUMENTATION

 1. Project Title

**Personal Expense Analytics Using Python**

 2. Objective

The main objective of this project is to analyze personal expenses using Python and visualize spending patterns through graphs and dashboards. The project also predicts future expenses using Machine Learning techniques.



 3. Problem Statement

Managing personal expenses manually is difficult and time-consuming. This project helps users:

* Track expenses
* Analyze spending habits
* Identify major expense categories
* Predict future expenses

using data analytics and visualization techniques.


 4. Technologies Used

| Technology   | Purpose                |
| ------------ | ---------------------- |
| Python       | Programming Language   |
| Pandas       | Data Analysis          |
| Matplotlib   | Data Visualization     |
| Seaborn      | Advanced Visualization |
| Scikit-learn | Machine Learning       |
| Streamlit    | Dashboard Development  |


 5. Dataset Used

Dataset Name:
**Personal Finance Dataset**

Dataset Source:

[Kaggle Dataset Link](https://www.kaggle.com/datasets/alhamdulliah123/personal-expense-analysis-and-budget-prediction?utm_source=chatgpt.com)

Dataset contains:

* Date
* Expense Category
* Amount
* Payment Mode
* Monthly Expenses


 6. Modules in Project

 a) Data Collection

The dataset is collected from Kaggle in CSV format.

b) Data Preprocessing

The following preprocessing steps are performed:

* Handling missing values
* Removing duplicate records
* Date conversion
* Creating month and year columns

 c) Data Analysis

Expense analysis performed:

* Monthly Expense Analysis
* Category Wise Expense Analysis
* Payment Mode Analysis
* Top Expenses Analysis

 d) Data Visualization

Visualizations created:

* Line Charts
* Bar Charts
* Pie Charts
* Histogram
* Heatmap

 e) Machine Learning Prediction

A Linear Regression model is used for:

* Expense Prediction
* Future Expense Forecasting

 f) Dashboard Creation

A Streamlit dashboard is developed to:

* Display dataset
* Show charts interactively
* Display predictions
